# How To Contribute

Thanks for contributing!

## Installation

- `git clone git@github.com:ember-template-lint/ember-template-lint-plugin-prettier.git`
- `cd ember-template-lint-plugin-prettier`
- `yarn install`

## Linting

- `yarn lint:js`
- `yarn lint:md`
